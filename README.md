# Capstone Assignment - Web Development

## 🚀 Project Overview
This is a web development capstone project built using React and Vite.  
It demonstrates modern frontend practices including component-based architecture and state management.

## 🛠 Tech Stack
- React
- Vite
- JavaScript (ES6+)
- CSS / Tailwind (if used)

## 📁 Folder Structure
src/
 ├── components/
 ├── pages/
 ├── context/
 ├── utils/
 ├── App.jsx
 ├── main.jsx

## ⚙️ Installation & Setup
1. Clone the repository  
   git clone <your-repo-url>

2. Install dependencies  
   npm install

3. Run the development server  
   npm run dev

## 📌 Features
- Dynamic UI with React
- Organized project structure
- Reusable components
